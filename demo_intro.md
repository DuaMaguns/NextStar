# Demo 简介

## 1.1 是什么

一款基于 AI 的大学生职业规划与学习路线生成网页应用，以"星际探索"为核心视觉隐喻，帮助学生从职业推荐、学习路线生成到求职准备，提供一站式职业规划服务。

## 1.2 面向谁

**核心用户**：准大学生、在校大学生（大一至大四），尤其是面临专业认知困惑、职业选择迷茫、缺乏清晰学习路径的学生群体。

## 1.3 主要功能

- **职业星系可视化**：以用户为中心恒星，推荐职业为周围行星，大小代表兴趣适配度，距离代表实现难度，直观展示职业选择空间
- **AI 驱动职业推荐**：基于学生的年级、专业、特长、爱好、MBTI 等多维度信息，智能推荐 10-12 个适配职业方向
- **个性化学习路线图**：生成从基础到进阶的树状学习路径，包含分支路线和细分职业终点，支持缩放、拖拽和平移交互


# Demo 创作思路

## 2.1 灵感来源

灵感源于学生群体普遍存在的职业迷茫现象——当专业前景、转专业、考公、就业和行业舆论交织在一起时，学生往往不知道下一步该往哪里走。我想做一款产品，把抽象的职业选择变成看得见、摸得着的"星际地图"，让每个学生都能找到属于自己的"星辰大海"。

## 2.2 想解决的问题

- **专业认知偏差**：学生对专业的多元发展方向了解不足，容易被单一就业评价或行业负面舆论影响专业自信
- **路径投入不透明**：同一专业可能通向就业、考研、考公、创业等不同路径，但每条路径的门槛、投入和收益缺少直观比较
- **学习路径碎片化**：学生知道目标职业后，不清楚需要学习哪些具体内容、按什么顺序学习，缺少系统的学习路线规划

## 2.3 为什么做这个方向

选择职业规划方向，主要基于以下判断和取舍：

- **用户需求刚性**：职业规划是每个学生必经的痛点，尤其在高校扩招、就业压力增大的背景下，需求更加迫切
- **技术可行性**：AI 技术的发展使得个性化职业推荐和学习路线生成成为可能，相比传统测评更具灵活性和准确性
- **可视化价值**：通过"星系"视觉隐喻，将抽象的职业规划转化为直观的视觉体验，降低理解门槛，提升用户参与感
- **轻量定位**：选择单页应用 + 轻后端的架构，避免复杂框架和数据库，便于部署和维护，聚焦核心功能而非技术炫技
